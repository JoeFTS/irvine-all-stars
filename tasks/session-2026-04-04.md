# Session Log — 2026-04-04

## What Was Done

### 1. Design System Redesign — "Premium Baseball Playful"
Complete design system overhaul across all 53 component/page files.

**Typography swap:**
- Dela Gothic One → hero/display headings (chunky, personality)
- Rubik → body/UI text (replaces Barlow everywhere)
- Oswald → stays for navbar, footer, structural UI

**Baseball-gear color palette (replaces old patriotic colors):**
- `--flag-blue: #0F1B2D` (deeper navy, was #002868)
- `--flag-red: #C8364B` (refined red, was #B22234)
- `--stitch-red: #E23D3D` (NEW — baseball seam accent)
- `--star-gold: #E2B74F` (warmer gold)
- `--leather: #C4956A` (NEW — glove tan)
- `--bat-wood: #8B6F47` (NEW — warm brown)
- `--grass: #4A7C59` (NEW — muted green for success states)
- `--dirt: #A68B6B` (NEW — infield dirt for input borders)
- `--cream: #F3EDE4` (old baseball feel)
- Card tint variants: `--tint-green`, `--tint-leather`, `--tint-cream`

**Decorative CSS utilities added:**
- `.baseball-stitch` / `.baseball-stitch-v` — dashed red stitch lines
- `.baseball-stitch-circle` — circular stitch decoration for heroes
- `.grain-overlay` — subtle noise texture on hero/dark sections
- `.diamond-accent` — rotated gold diamond shape
- `.badge-home-plate` — clip-path badge
- `.btn-stitch` — dashed border button variant

**Motion updates:**
- `hoverBounce` — scale(1.04) with spring easing (public page buttons)
- `hoverCard` — scale(1.02) + rotate(1deg) (card hover)
- `hoverStar` — rotate(15deg) for star icon hover
- Stagger timing increased from 0.06s to 0.08s

**Border radius overhaul (all pages):**
- Cards: `rounded-lg` → `rounded-2xl`
- Buttons: `rounded-md` → `rounded-full` (pill shape)
- Inputs: `rounded-md/lg` → `rounded-xl`

**Scope rules:**
- Public pages: full baseball treatment (hero fonts, stitches, grain, card tints, bouncy hover)
- Portal pages (admin/coach/parent): fonts + palette only, no decorative elements
