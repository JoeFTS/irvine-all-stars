### 2. Score Sheet Endpoint Fixes
- **RLS bug fixed**: The `/api/score-sheet` endpoint was creating unauthenticated Supabase clients for data queries, returning 0 players due to RLS. Fixed by using a service role client (`SUPABASE_SERVICE_ROLE_KEY`) for data queries after auth verification.
- **Deployment bug found**: Local edits weren't being committed, so `git pull` on VPS kept deploying old code. Root cause of multiple failed deploys.
- **PM2 env var issue**: `pm2 restart` reuses old environment — must do full `pm2 stop && pm2 delete && pm2 start` to pick up new env vars from `.env.local`.
- Added `SUPABASE_SERVICE_ROLE_KEY` to both local and VPS `.env.local`.
